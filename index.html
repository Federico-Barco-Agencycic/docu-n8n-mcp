📖 Usage Examples

JavaScript/Node.js

const response = await fetch('https://federico-barco-agencycic.github.io/docu-n8n-mcp/api/index.json');
const data = await response.json();
console.log(`Found ${data.meta.total_nodes} nodes`);

Python

import requests
response = requests.get('https://federico-barco-agencycic.github.io/docu-n8n-mcp/api/index.json')
data = response.json()
print(f"Found {data['meta']['total_nodes']} nodes")

PowerShell

$data = Invoke-RestMethod -Uri 'https://federico-barco-agencycic.github.io/docu-n8n-mcp/api/index.json'
Write-Host "Found $($data.meta.total_nodes) nodes"
